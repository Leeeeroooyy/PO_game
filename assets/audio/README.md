Put your audio files in these folders. The game checks the same base name with .ogg, .wav, or .mp3.

Music:
- music/main_menu_theme.ogg
- music/gameplay_theme.ogg
- music/enemy_structure_fight_theme.ogg
- music/dead_theme.ogg

Sound effects:
- sfx/attack_melee.ogg
- sfx/attack_ranged.ogg
- sfx/attack_tower.ogg

Recommended format: OGG Vorbis for music, WAV or OGG for short sound effects.
